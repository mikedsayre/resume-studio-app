# Welcome to Resume Studio!

![Resume Studio Mascot](images/resume-studio-logo-256px.png)

Resume Studio helps you craft professional resumes with ease using Markdown for content and custom CSS for styling. See your changes instantly in the live preview!

## How to Use

### üìù Markdown Editor
*   **Write Content**: Use standard Markdown syntax (headings, lists, bold, italics, links) in the "Markdown" tab to structure your resume.
*   **Tips**:
    *   Use `#` for your name (H1).
    *   `##` for sections like "Experience" or "Education" (H2).
    *   `*` or `-` for bulleted lists.
    *   `**text**` for bold, `*text*` for italics.
    *   `[Link Text](URL)` for links.

### üé® CSS Editor
*   **Style Your Resume**: Switch to the "CSS" tab to write custom CSS rules. These styles apply *only* to your resume preview, ensuring no interference with the app's interface.
*   **Elements**: Target standard HTML elements (`body`, `h1`, `p`, `ul`, `li`, `a`) or specific IDs/classes you add in your Markdown.
*   **Example**: Change font, colors, margins, or add borders.

*   **Import**: Click the "Import" button, then choose "Markdown (.md)" or "CSS (.css)" to load an existing file from your computer into the respective editor.
*   **Export**: Click the "Export" button, then choose "Markdown (.md)" or "CSS (.css)" to download your current editor content as a file for backup or sharing.

### üìÅ Templates
*   Opens a modal where you can:
